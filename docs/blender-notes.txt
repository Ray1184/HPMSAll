-------------------------------------------------------------------------------------------------------------------------------------------

Zgl     -> -Yb
RXgl    -> 90 – RXb
RYgl    -> – RZb
RZgl    -> Ryb

-------------------------------------------------------------------------------------------------------------------------------------------
MASCHERE DI PROFONDITÀ
-------------------------------------------------------------------------------------------------------------------------------------------
Per generare maschere di profondità semplici procedere come segue:

- Creare un cubo che copre la parte della mesh di cui si vuole esportare la profondità;
- Aggiungere un Modifier Boolean con intersezione fast per tenere solo la parte sovrapposta;
- Una volta create tutte le maschere usare il Modifier Decimate per ridurre la dimensione delle mesh di prfondità.
